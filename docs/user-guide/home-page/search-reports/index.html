Search Reports

You can search for all the reports across the BI Platform in the Search box located at the top of every section.

The searching parameters that you use are Report Name, Report Description, Path, Hub Folders, and User Tags.

As you start typing in the search box, the report names that completely/partially match with the text you enter get listed. If you select any of the tags listed, then all the reports that are tagged with the name will be listed.

Searched reports

The Reports are categorized as Accessible and Inaccessible with a count of the reports in the brackets.

You can view the Accessible reports, and request access for the Inaccessible (accessible by other users) reports.

Advanced Search Reports#

Use the Advanced search feature for a more precise report search.

You can search reports based on the following parameters:

  1. Report Name: Enter the complete report's name or part of (regex) the report's name and select Search.
  2. Report Description: Search for a text description that appears within the report(s).
  3. Report Owner: Name of the report's owner.
  4. Platform: Reports based on platforms (Agents) configured in BI Hub.
  5. Report Tags: Search tagged reports. When you click in the Report Tags box, a list of all the tag keywords as set by admin are listed. You can select more than one tag as appropriate for your search.
  6. Custom Attribute: Reports based on default custom attribute as set by the admin and also user-defined attributes. You can select More at the top-right corner to show/hide the attributes on the reports.

Based on your above search criteria appropriate search reports will be listed.

To resume the default search criteria to show all the reports, select Reset.
