Settings

On the Settings page, you can customize your BI Hub user profile. Following are the settings that you can configure.

User Settings Page

Profile#

In the Profile section, you can edit your email id, division, and edit/delete the user profile photo.

You cannot edit the username.

Customization Settings#

Here you can enable/disable the following options:

  • Open Report in new tab: If enabled, the reports will open in a new tab in the browser. This setting is Off by default. When the user enables this setting, that takes precedence over the default one.
  • Email Notification: If enabled, you will receive the notifications in email.
  • In-App Notification: If enabled, the notifications will pop up in the BI Hub application.

Authorize Agents#

As an additional security measure, some of the agents in the BI Hub platform have been configured with both admin and user side authentication.

You can only access the reports for the listed agents in this section once you validate your sign-in credentials. Select Login next to the agent(s).

Microsoft Sign-in screen

Enter the user mapping credentials for the agent.

Success credential verification screen

Troubleshooting#

If there is a mismatch in the platform user mapping and the user credentials, then the below authentication error will be shown:

Check the Microsoft Office login credentials for any other user log-in. You must log out the other user and sign-in as your logged-in user credentials.

If you are still unable to sign in, please contact your BI Hub admin.

Preferences#

You can set the Landing Page by selecting the tabs from the drop-down list.

You will land on the preferred page chosen above the next time you log in to your profile.

This is a user-customizable setting and personal to each user.
