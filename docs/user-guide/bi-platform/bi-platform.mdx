In the **BI Platform** section, you can view all the reports categorized agent-wise.
You can only access the reports that the BI Hub admin has permitted you to.

You can also view the Custom URLs created by BI Hub admin on this page.

  <div class="center">
    <Zoom>
      <img alt="BI Platform" src={useBaseUrl('doc-images/user-guide/bi1.png')}/>
    </Zoom>
  </div>

You can navigate to the report(s) to view them and also add the report(s) to "My Folders".

You can also perform the common report's actions as explained in the <Link to={useBaseUrl('docs/user-guide/reports#user-actions-on-the-report')}>User Actions on the report</Link> section.
