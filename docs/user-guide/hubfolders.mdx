In the **Hub Folder** section, as a user, you can view the folders/reports organized by BI Hub admin.
This serves as a general Hub for all the users to view the common reports organized by the admin.

You are allowed to also select and add the folders/reports to **My Folders** section.

:::note
You cannot add/delete/edit the folders here.
:::
