Uninstallation of BI Hub in Windows Platform

Follow these steps to uninstall any component of BI Hub from the system:

  1. Navigate to the folder where the BI Hub component is installed.
  2. Locate the Uninstaller application uninstall.exe and run as administrator.
  3. Click Yes on the Confirmation dialog box.
  4. Click OK on the pop-up window with the message - "BI Hub component has been uninstalled".
  5. If the installer prompts for the restart, kindly restart the server. Donâ€™t install anything else without restarting.
  6. There will be residual files generated by the application. You can either keep the files or delete them.
