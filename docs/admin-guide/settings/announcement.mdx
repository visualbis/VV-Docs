To announce important information on user's BI Hub page, an admin must create and manage the **Settings > Announcement** section.

  <div class="center">
    <Zoom>
      <img alt="All Announcements" src={useBaseUrl('doc-images/admin-guide/settings/announcement-lists.png')}/>
    </Zoom>
  </div>

* Select **Add Announcement**.

  <div class="center">
    <Zoom>
      <img alt="Announcement Tab" src={useBaseUrl('doc-images/admin-guide/settings/announcement.png')}/>
    </Zoom>
    <p>Add a new announcement message</p>
  </div>

* In the pop-up window, enter the following details:
  
  * Title: Title of the announcement pop-up
  * Description: Further information about the announcement.
  * Type: Alert, Warning or Information.
  * Dismissible: Set the toggle ON/OFF to allow/deny the user to dismiss the information.
  * From: Date-timestamp to trigger the announcement.
  * To: Set the expiry date-timestamp for the announcement.
  * Select Users: Select the all or selected users and groups to share the information with.

* Select **Update** to save your changes.

The users will be notified by your announcements on their homepage screen.
