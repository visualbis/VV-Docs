Settings

The Settings section allows an admin to configure BI Hub for all users.

BI Hub settings

Following are the tabs available:

  1. General: Configure general application settings.
  2. Report: Report metadata attributes.
  3. Theme: Theme configurations for BI Hub portal.
  4. User Dashboard: Customize User Dashboard and Custom Dashboard.
  5. Announcements: Add important announcements on the user's BI Hub page.
  6. Email Template: Edit and configure Email templates.
