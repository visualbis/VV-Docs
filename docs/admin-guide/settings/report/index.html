Report Tab

In addition to the metadata from BI platform, we can have our own metadata for reports. From the Settings > Report section, you can manage the Custom Fields for the reports.

Report Tab with default Attributes

You cannot edit or delete the default attributes. However, you can only edit the Certification attribute but still cannot delete it.

Add Attributes#

You can also add your own attributes on a report other than the default Custom attributes. The additional attributes that you add are called user-defined Custom attributes.

  • Select Add Attribute and enter the field values.

    Add Custom Attribute form

  • Enter a Name for the metadata field, Label, Description, and the Type of Attribute.
Field NameDescription
NameAttribute field name
LabelCustom Field Name to be displayed
DescriptionTooltip text description for the field
TypeType of Attribute:
  • Text: Any text(string) value
  • URL: URL link type
  • Date: Date range
  • List: drop-down menu with a list of Labels and their values
  • User Selected
  • Select Submit.

Below is a screenshot with all the new attributes added:

  • Select an attribute(s) (non-default) and select Edit to edit the properties.
  • Select and attribute(s) (non-default) and select Delete. This will delete the all the report attributes which cannot be retrieved once deleted.
  • To enable/disable the attribute in the Report details screen, use the ON/OFF toggle against each attribute.

Map Attributes to Report(s)#

  • Go to Reports > Platform Folders.
  • Enter the values for the Custom Fields

The users can now view the custom fields that you added to the report from BI Platform.
