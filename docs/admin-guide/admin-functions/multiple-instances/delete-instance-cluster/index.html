Deleting instances of cluster

Instances of cluster are same as other agent instances and the Agent Manager > Instance Manager lists those instances, with the primary instance marked with an asterisk. BI Hub enables the administrator to delete the instances and/or add new instances to the cluster. The deletion step happens in the Agent Manager > Instance Manager by selecting the instance and clicking delete and adding new instances to the cluster happens in Cluster Manager as indicated in Section Modifying Primary instance of a cluster. If the Primary is deleted, BI Hub automatically makes the secondary instance as Primary and it is the responsibility of the administrator to add a second instance to the cluster.
