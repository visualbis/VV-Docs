Change Administrator Password

  1. Go to C:\Program Files\Visual BI Solutions\VBI View\VBIViewServer\utilities.
  2. Right-click on BI Hub Change Admin Password.exe and select Run as administrator.

    Admin change password utility

  3. Select Yes to continue with the change password utility.
  4. In the Change Password utility, a prompt to enter the new password appears. Type the new password and press ENTER.

    Enter the new password prompt

  5. Confirm the password, and your Admin password is now reset.
