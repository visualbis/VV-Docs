The application log contains events that happen in the application.

There are three Logger Levels:

1. **Info**: Represents all information messages that capture the events occurring during the usage of the application
1. **Error**: Represents error events that occur during the application
1. **Warning**: Represents warning messages

## Server Logs and Configuration

To change the logger level in the Server, go to the location `C:\Program Files\Visual BI Solutions\VBI View\VBIViewServer`. Right-click on the file `BI Hub Server Config` and click **Run as administrator**.

## Agents Logs and Configuration

To change the logger level for agents, go to the `<agent installation directory>`.

For example change location to `C:\Program Files\Visual BI Solutions\VBI View\TableauAgent` for Tableau agent, and edit the `manager.ini` file.

```json title="<agent installation directory>\manager.ini"
[Log]
log level = INFO //ERROR or WARNING
```

Set the 'log level' property to 'INFO', 'ERROR', or 'WARNING'.

:::note
Restart the instances/ services for the log level changes to take place
:::
