This section is used by an admin for managing users and groups. An admin can manage group permissions and also define power users in this section.

The screen displays the list of user **Names**, **Agents** configured for each user, and the **Authentication** type.

The user count mentioned in parenthesis next to the User Manager Label (27) represents the total number of users existing in BI Hub.

You can filter the users based on User Names/Agent(s) names, or Authentication types:
