User Manager

This section can be used by an Administrator to add users manually from Microsoft AD to BI Hub. Click on User Manager from the application pane.

Add New User#

  1. Click on Add User on the right top as shown below.

    Add User

  2. Enter Fullname, and select:

    • Native to add a user who will be authenticated by BI Hub (because the user is currently authenticated by BI platforms and not AD). OR
    • Windows AD to add a user who will be authenticated by Windows AD.

    Authentication details

  3. Click Continue.

  4. Select the agent to be added for the user and the corresponding username. Click Add to add more agents.

  5. Click on Submit.

  6. The added user will be reflected in the User Manager Page.

  7. In order to synchronize reports for this user, one of the following actions can be performed

  • Wait till the next scheduled synchronization activity happens.
  • Follow steps in Section Synchronize Reports of Users to Sync Reports with a single click
  • Manually add reports to the user using User Manager as explained in section Add Reports to an existing user.
