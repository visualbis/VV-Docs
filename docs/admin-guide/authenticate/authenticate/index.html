Edit Microsoft AD details in BI Hub#

The admin can edit Microsoft AD details using the following:

  1. Click on Authentication Menu from the application pane.
  2. Select the appropriate row and click on "Edit" to launch the Edit Authentication page.
  3. Modify the details appropriately.
  4. Click Save when the modifications are completed.
